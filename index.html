BronxConnect AI

Problem

Many people in the Bronx struggle to find support services like housing, food, and mental health resources. The process can be confusing and slow, which delays getting help.

AI Solution

BronxConnect AI is a system that connects users to the right services by asking simple questions and giving fast recommendations.

Implementation

This can be built using a website with AI tools that collect user input and match it with available services.

Impact

This helps people get support faster, reduces stress, and improves efficiency for organizations.

Ethics & Challenges

User privacy must be protected and the system must avoid bias in recommendations.

Reflection

This project helped me understand how AI can be used to solve real problems and how to build a simple website.
